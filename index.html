Селекторы

CSS

виды селекторов

По тегу: h1 { color: green; } - присваевает стиль определенному указаному тегу

По атрибуту: [title="да, да уже CSS"] { border: 10px solid red; } - присваевает стиль определенному указаному атрибуту

по id: #lgn { background-color: blue; } - присваевает стиль определенному указаному id

по классу: .attention { color: blueviolet; } - присваевает стиль определенному указаному классу

смешанный: p#title.attention { font-style: italic; } - - присваевает стиль указзанным нескольким селекторам
